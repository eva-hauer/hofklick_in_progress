Inbetriebnahme von Hofklick
1.	Repository klonen: git clone https://github.com/eva-hauer/hofklick_in_progress
2.	Projekt öffnen: Projektordner im Code-Editor (z.B. WebStorm) öffnen.
3.	Seite öffnen: Datei index.html über den integrierten lokalen Server (Browser-Symbol) öffnen.
4.	Mobile Ansicht testen: Die Entwicklertools (F12) öffnen und die Geräte-Simulation für ein Smartphone aktivieren, um das Layout optimal zu betrachten.

oder einfach www.haua.at/hofklick besuchen und über die Browser-Adresszeile installieren.
